Frequently Asked Questions

Some key questions for our SDK and solutions are listed here.

What is augmented reality?

Augmented reality is a live direct or indirect view of a physical, real-world environment, whose elements are augmented by computer-generated sensory input such as sound, video, graphics or GPS data.

Can you get direct access to the depth and RGB sensor data Meta uses for making custom vision plugins?

Yes. See the DepthSense SDK for a C++ plugin writing guide.

The camera is having issues detecting hand input. How do I fix this?

There are a few known causes of interference and limitations that can prevent the camera from accurately detecting hand gestures. Some of these include highly reflective jewelry on the hand, wrist and lower forearms, long sleeve tops and loosely fitted sleeves. For more detailed information, please refer to the Troubleshooting Guide for more details.

The camera is having issues detecting certain gestures. How do I fix this?

There are actions that the camera rely on to recognize that a specific gesture is being performed.

  • For a grab to be reliably registered, the user must fully close their hand into a fist.
  • For a pinch to be reliably registered, the user must touch the tip of their thumb to the tip of their index finger so that there is a visible circular hole.

Please refer to the Troubleshooting Guide for more details.
