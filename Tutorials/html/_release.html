  • DOWNLOADS

Upgrading and Requirements

If you are upgrading a project made with an earlier SDK version, you should familiarize yourself with the release notes and duplicate your project before attempting to import the newest SDK package. Make sure everything works as before by testing your App functionality after updating and report any upgrade difficulties you encounter to our Dev Center Forums

See Requirements to make sure you have what you need to use the SDK.

Version Naming

All Meta SDK versions have 4 numbers separated by decimals. Each decimal place has it's own unique meaning for you to decipher the SDK release.

Decimal Place Meaning
First Major version, coinciding with new hardware support
Second Minor version, introducing new major features
Third Revision number, fixing and tweaking existing features
Fourth Internal build, typically not given to public for internal use

FEATURES AND IMPROVEMENTS

  • Touch Dwell time per MetaBody can be set in the Inspector Window
  • Touch Dwell Indicators now can be disabled in code with the following line:
    • InputIndicators.Instance.dwellIndicators = false;

CHANGES

  • IMU raw sensor data can now be accessed through the use of MetaLocalization.ImuLocalizer in code. The varables made public :
    • accerlometerValues,
    • gyroscopeValues,
    • magnetometerValues and
    • imuOrientation
  • Documentation Namespaces and Classes cleaned up and now no longer contain:
  • Deprecated/ Internal Classes

FIXES

  • Meta Keyboard now supports the use of typing on a physical keyboard for input.
  • Meta Keyboard positioning adjusted to correctly appear underneath the input field regardless of the position of the input field.
  • MGUI Text Input caret is now visible.

KNOWN ISSUES

  • Meta.unitypackage is compatible only with Unity 4.6.0 version
  • Some links may be broken in MetaBehaviour Scripts for offline documentation
  • Meta Apps may crash if they do not have Meta 1 Glasses plugged in during play
  • MouseClick navigation only works in Mono-Mode during play
  • Unity:
    • VectorMesh is independent of GameObjects and may cause a disjointed effect during rapid movement
    • X-Ray Shader flickers between opaque and transparent

Meta SDK 1.0.1

  • By installing or using the Meta SDK, you agree to be bound by the terms specified in the Meta Developer Kit End User License Agreement.

IMPORTANT NOTES

  • If you have downloaded Meta Installer 1.0.0 prior to October 21 2014, updating your Unity project by re-importing a new Meta.unitypackage after the 10/21/2014 will break MetaBody script
  • Meta SDK 1.0.0 is the first production level ready Development SDK released for public usage.
  • By installing the Meta SDK, you agree to the terms specified in the Meta Developer Kit End User License Agreement.
  • Always duplicate the project before attempting an upgrade import.
  • Meta functionality is now contained within Meta.dll and Meta-Editor.dll.

FEATURES AND IMPROVEMENTS

  • Grab to rotate and scale are now options in MetaBody
    • Added two-handed grab to move, rotate and scale
  • Touch Dwell indicator enabled by default on MetaBody.touchable Game objects
  • Modified MGUI objects to only be interactable when visible
  • UI elements can not be interacted with while the hand is grabbing or pinching objects (or up to 0.5 seconds after the hand has finished grabbing or pinching).
  • Significantly improved smoothness of pinch to move and pinch to scale
  • MetaBehaviour scripting tutorial updated in the documentation

CHANGES

  • Default Touch Dwell time to 1 second
  • Default Palm and Finger locking hand type changed from Right to Either
  • Pinch to move and pinch to scale made mutually exclusive
  • MetaBehaviour is now part of namespace Meta
    • include the following line when accessing MetaBehaviour functions
      using UnityEngine;
      using Systems.Collections;
      using Meta;

FIXES

  • MarkerDector is public, allowing you to use:
    • Meta.MarkerDetector.Instance.markerSizeMeters,
    • Meta.MarkerDetector.Instance.updatedMarkerTransforms and
    • Meta.MarkerDetector.Instance.GetNumberOfVisibleMarkers()
  • NullReferenceError when the scene has a MetaBody with MarkerTarget checked and your Meta device is not plugged in
  • Meta.MarkerTargetIndicator.indicatorsVisible not working
  • Fixed MarkerPlaceability not working
  • CameraTextureSource to only be enabled when RGB mode (F8) is on, saving a lot of CPU power when it is off
  • VectorMesh and GameObjects are now in sync
  • X-Ray Shader now fixed to opaque in Meta primitives

KNOWN ISSUES

  • Meta Apps may crash if they do not have Meta 1 Glasses plugged in during play
  • MouseClick navigation only works in Mono-Mode during play
  • Unity:
    • HUD Layer objects may be overlapped to the IMU loading screen if there are more than one GameObjects locked to HUD
    • Raw IMU values not available
