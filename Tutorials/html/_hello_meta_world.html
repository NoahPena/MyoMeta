Hello MetaWorld

Your first App using MetaWorld and MetaAppObject

Overview

This guide will show you how to create your first Meta application: a cube that you can view through the glasses.

By the end of this guide, you will be able to:

  • Know how to import the Meta project
  • Create a Unity scene
  • Add the MetaWorld prefab
  • Use an App object to manage your App
  • Add a MetaCube primitive
  • Run a Meta App from within the Unity Editor

Requirements

  • Unity for Windows
  • Meta SDK Installer
  • One pair of Meta glasses

This tutorial also assumes that you have already completed the following:

  • Setup Guide

Steps

Setting up

  1. Plug in and set up your Meta glasses
  2. Run Unity

Creating a new Meta project

  1. Create a new Unity project and import the Meta package
    1. If you don't see the Project Wizard, select Menu Bar > File > New Project
    2. Check the box that says @ref MetaDotunitypackage
      Meta.unitypackage in the Project Wizard
    • Alternativly, you can add it to a pre-existing project by: Assets> Import Assets> Meta
  2. Create a new scene named 'Hello MetaWorld'
    1. Menu Bar > File > New Scene
    2. Save the new scene as 'Hello MetaWorld'

Adding MetaWorld to the scene

  1. Delete 'Main Camera' from the scene
    1. Select 'Main Camera' in the Hierarchy window
    2. Right click > Delete
  2. Click and drag the MetaWorld prefab from the Project View to the Hierarchy window at position (0, 0, 0)
    1. Project View > Assets > Meta > MetaCore > Prefabs > MetaWorld
    2. Modify the values in the Inspector window so that the position is (0, 0, 0)
MetaWorld in your scene

Adding an app object with the MetaAppObject script

  1. Create an empty GameObject at position (0, 0, 0)
    1. GameObject > Create Empty
    2. Modify the values in the Inspector window so that the position is (0, 0, 0)
  2. Rename the empty GameObject to 'Hello MetaWorld App'
    1. Select the empty GameObject in the Hierarchy window
    2. Left-click the empty GameObject again in the Hierarchy window to get an editable text field
    3. Type in 'Hello MetaWorld App'
  3. Add the MetaAppObject script to your 'Hello MetaWorld App' GameObject
    1. Select the 'Hello MetaWorld App' GameObject in the Hierarchy window
    2. Click Add Component at the bottom of the Inspector window
    3. Type in 'MetaAppObject' and click on the search result to add it
Hello MetaWorldApp in your scene

Adding a MetaCube primitive prefab

  1. Click and drag the MetaCube prefab from the Project View to your 'Hello MetaWorld App' GameObject in the Hierarchy window at position (0, 0, 0.4)
    1. Search for MetaCube in the search box at the top-right of the Project View OR navigate to Assets > Meta > MetaSpace > Resources > _Prefabs > MetaPrimitives > MetaCube
    2. Click and drag the MetaCube prefab from the Project View to your 'Hello MetaWorld App' GameObject in the Hierarchy window
    3. Modify the values in the Inspector window so that the position is (0, 0, 0.4)
      Dragging the MetaCube primitive prefab
      MetaCube in your scene
  2. Check that the "Maximize on Play" button is selected in the Game View and that your Unity Editor window is maximized
  3. Save the scene
  4. Put on your Meta glasses
  5. Click the Play button to run the scene
  6. Turn your head to look for the cube (or push F4 to recentre the Game View)
    1. If you are not seeing 2 side-by-side images on your computer monitor, press F2 to switch to side-by-side 3D mode
    2. If you are not seeing a 3D image through the glasses, press the 3D toggle button on the bottom left of your Meta 1 Control Box
    3. If you are seeing a very small image through the glasses, press F3 to switch to a rectified view
  7. Grab the cube and move it around with your hand!
    1. Lift up your open hand in front of the glasses to the position that you see the cube
    2. Grab the cube by closing your hand
    3. Move your closed hand in varying directions to move the cube
MetaCube in the Game View

Toggle the live camera display

  1. Press F8 to toggle a live RGB camera feed of the physical world that appears behind your augmented content.
    1. For a full list of keyboard shortcuts, refer to the Keyboard Shortcuts page.

Conclusion

Congratulations, you have made your first Meta app with a grabbable MetaCube primitive! Every Meta app that you create will need the 'Main Camera' object to be deleted and the MetaWorld prefab to be inserted in order to work with the Meta glasses. The contents of your app should also be stored within your App GameObject - in this case, 'Hello MetaWorld App'.

For more information about MetaWorld, refer to the MetaWorld Prefab section of the Meta App Programming Guide.
