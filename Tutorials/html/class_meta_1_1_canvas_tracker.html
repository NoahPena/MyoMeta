Meta.CanvasTracker Class Reference

Description

CanvasTracker can detect and track 3D rectangles on a scene.

CanvasTracker v1.0

CanvasTracker can detect and track 3D rectangles on a scene. For example, it can detect tops of boxes, books, any rectangular surface. For more details and usage, see Meta's SDK guide.

Object Placement on Canvas

When a surface is horizontal (such as a box-top), the Canvas-tracked object will have its z-axis facing the edge of the surface that is closest to the user. When a surface is vertical (such as a box-wall), the Canvas-tracked object will have its z-axis facing down and its y-axis facing the user.
