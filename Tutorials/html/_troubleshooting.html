Troubleshooting Guide

Troubleshooting

Meta depends on both hardware and software components to work together for the full experience. If one or more of these components are causing you trouble, it may be a simple issue. We will walk you step-by-step on how to configure and troubleshoot both hardware and software systems.

  • Software
  • Hardware

Software

Installation Issues

Meta executable is getting stuck on the 'Starting your Meta 1' Loading Screen

This issue may arise if the device is not plugged in properly. Check to see that the device is connected to the computer.

If you continue to experience this issue, go into the device manager

  • Control Panel> Devices and Printers> Device Manager

Check you have both Imaging Devices> DepthSense and COMS&PORTS connection.

Successful Connection in Device Manager

If you do not notice both of the devices registered do a clean uninstall and re-install. If you continue to experience unsuccessful Device registration issues it may be due to the driver connection issues on your machine.

Please refer to the following link: FTDI Driver Install Guide

Not getting Gesture recognition in executable

This issue commonly happens when you have ghost processes running in the background. To remove the background processes that may be interfering with the Camera, open up Task manager and check through the list of processes to ensure there is no more than one Meta app running.

IMU Driver Issue (COMS&PORTS is not being registered in the Device Manager)

Manually install FTDI drivers FTDI Driver Install Guide

Refer to IMU troubleshooting IMU Issues

SoftKinetic DepthSense SDK wants to uninstall my previous version

During the installation process, the SoftKinetic DepthSense SDK may ask you to uninstall a previous version you have on your computer. If your version differs from the one that is being installed, you should allow it to uninstall the previous version and install the current supported version of the DepthSense SDK.

Microsoft Visual C++ 2010 Redistributable wants to quit applications that are already using it

During the installation process, the Microsoft Visual C++ 2010 Redistributable may ask you to quit applications that are already using it. Click Ignore to continue the installation.

Unity/Meta SDK Issues

Meta.exe Error: Could not preload global game manager #0, i=0

During the installation process, the installer asks you to install the SoftKinetic DepthSense SDK and a Microsoft Visual C++ Redistributable. If you have not installed both of these components along with our Meta SDK, or if your computer does not have the correct versions, it may lead to an error when launching Meta.exe. Please re-install the Meta SDK and install the additional components when prompted.

Windows 7 issues

The Meta SDK has only been tested in a Windows 8/8.1 64-bit environment. Some of Meta SDK components may not be compatible with Windows 7 and may experience issues. We recommend that you upgrade to a Windows 8 or higher 64-bit environment before using the Meta SDK.

DepthSenseServer.exe Error: MFPlat.DLL is missing from your computer

On the 'N' and 'KN' versions of Windows (available in some parts of Europe), the Media Feature Pack and its associated DLL files are not installed by default in Windows.

To fix this, download and install the Media Feature Pack for your version of Windows:

  • Windows 8.1 N/KN
  • Windows 8 N/KN
  • Windows 7 N/KN with Service Pack 1

Problems recognizing hand gestures in Unity

There are a few known causes of interference and limitations that can prevent the camera from accurately detecting hand gestures. These include:

  • Objects close to the user's hands will appear to be part of the user's hands in the system.
  • The hand segmenter will choose the two closest objects to the camera, within the configured range of the hand input system, as the user's hands.
  • Wearing highly reflective or dark-coloured jewellery on the hands, wrist and lower arm can negatively interfere with the hand segmenter.
  • Wearing loose long sleeve tops can negatively interfere with the grab gesture.

If you are having problems with getting a particular gesture recongized, please try the following:

  • Grab: the user must fully close their hand into a fist
  • Pinch: the user must touch the tip of their thumb to the tip of their index finger so that there is a visible circular hole

If you are not getting any hand detection at all, refer to the Camera Issues section below.

Problems with head tracking movement detection

Currently we are supporting both Serial and HID IMUs. If the software IMU setting does not match with the IMU model in the glasses you are using, it will not be able to perform head tracking properly. To change this setting, refer to User Settings.

Also, see the IMU Issues section below.

User Settings are not being saved correctly

If you are finding that your User Settings are being reset after every time you run your Unity scene, your User Profile may have been corrupted. To fix this:

  1. Delete the Meta\User Profiles folder in your My Documents folder.
  2. In your Meta project, search for .userProfile in your Project panel, and delete the profiles that you find.

This will allow Meta to recreate your user profile the next time you open the User Settings Window, or run a Meta Unity scene. You should then set your IMU model again, either in the User Settings Window (Meta menu>User Settings Window), or by opening Meta.exe or the MetaWelcomeScene.unity in the Meta project.

Unity is displaying compile errors on code that should be working

The Meta SDK officially supports Unity 4.5 and higher. If you are sure that all your code is correct, but Unity is displaying errors and your application is not running correctly, try triggering a recompile of the Unity project by editing a script and saving it.

If that doesn't work, try exiting and reopening Unity, or try rebooting your computer.

Hardware

Display Issues

Glasses display flashes white and black (early dev kit only)

If your glasses appear to be flashing white and black after being turned on and connected to your computer, there may be an issue with the HDMI conversion box. To fix this issue, locate the area of the Meta Control Box where the HDMI cable connects to and check the following:

  1. Make sure that you have connected the Mini USB cable to a power source.
  2. Press the small, round button next to the HDMI cable connector.
    • This will switch the display mode, and should cause the glasses to stop flashing.

Control Box Issues

Control Box flashes a red light when trying to turn it on (early dev kit only)

If your Meta Control Box is flashing a red light when you are trying to turn it on, it means that the Control Box is out of battery. Connect the Micro USB cable into the Control Box and a power source (ideally a power outlet) and allow the box to charge for at least 15 minutes.

See Control Box Features for further information.

Camera Issues

If the camera is not working: Diagnosing DepthSense problems

If you are not seeing camera input working properly in Unity, you may need to check DepthSenseViewer to see if Windows is detecting the camera correctly.

  1. Make sure the USB cable for the camera is plugged in correctly.
  2. Open the Start Menu by pressing your Windows Key
  3. Type in DepthSenseViewer and click on the search result
  4. In DepthSenseViewer, under Devices on the left, you should see DS325.
  5. Click 'Color Node' and click the 'Reg' button in the toolbar.
  6. Double click on 'Color' to make the Color window appear.
  7. Click Play button in the toolbar.
  8. You should see a color camera feed.

If you do not see a camera feed, you can try the following steps to attempt to fix any software problems:

  • Try plugging the USB cable for the camera into another USB port.
  • Reinstall the DepthSense SDK by uninstalling and reinstalling the Meta SDK Installer.

If you do see a camera feed, please follow this flowchart to troubleshoot your issue:

DepthSense camera troubleshooting flowchart

To learn more about how to set up your Meta 1 hardware and for troubleshooting, please visit Meta Support.
