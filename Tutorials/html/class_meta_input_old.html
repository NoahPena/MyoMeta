MetaInputOld Class Reference

Description

MetaInputOld provides hand, gestureOld and reticle-based input from the depth camera and IMU.

To learn how to design interfaces for use with Meta Input, see Meta Human Interface Guidelines.

To learn more about how to implement gestureOld input, see Meta App Programming Guide.

MetaInputOld supports multiple forms of input.

  • Fingers: Detect that a finger touches a cube.
  • Gaze: Detect that the center of someone's screen is pointing at an object.

For example, suppose you want to know if a finger has touched an object:

void Update ()
{
if ( MetaInputOld.ContainsFingers(gameObject.collider) )
{
Debug.Log("Touched");
}
}
