Meta.CanvasTarget Class Reference

Description

Attach this script to objects in order to control them through the CanvasTracker.

Notes / Features

  • Please use default setting for Plane Type (SmartHorizontal). Other modes might be undefined.
  • The CanvasTarget comes with motion filtering functionalities. There is simple Kalman Filtering, which is enabled by setting m_kalmanFiltering=true. The amount of smoothing is controlled by m_KalmanVelocity./
  • Filtering will introduce some lag.
  • For more details about Canvas Tracking, check out CanvasTracker.cs and the tutorials available. enable other modes.
