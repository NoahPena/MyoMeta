Meta.MetaBehaviour Class Reference

Description

MetaBehaviour inherits from MonoBehaviour and adds augmented reality behaviour for use with Meta.

To make a script work with Meta, change the : MonoBehaviour inheritance of the class to : MetaBehaviour instead. MetaBehaviours add themselves to MetaManager.MetaObjects list.

MetaBehaviours require a MetaBody to initiate their events and when you add a MetaBehaviour to a game object, a MetaBody will be automatically added if none is present.

MetaBehaviour's handler methods can be implemented by your scripts.

For example, here is example code for touch events from the MetaSDKGuideApp :

public class OnTouchEnterTestScript : MetaBehaviour {
public void OnTouchEnter()
{
renderer.material.color = Color.red;
Debug.Log("OnTouchEnter");
}
public void OnTouchHold()
{
renderer.material.color = Color.green;
}
public void OnTouchDwell()
{
renderer.material.color = Color.cyan;
}
public void OnTouchExit()
{
renderer.material.color = Color.blue;
}
}

/// Grabbable example:

public class OnGrabTestScript : MetaBehaviour
{
public void OnGrab()
{
renderer.material.color = Color.red;
}
public void OnHold()
{
renderer.material.color = Color.green;
}
public void OnRelease()
{
renderer.material.color = Color.blue;
}
}
