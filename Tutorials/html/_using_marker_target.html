Tracking a MarkerTarget

Overview

This guide will show you how to use the Marker Target option in MetaBody to track GameObjects to a printed marker in the real world.

By the end of this guide, you will be able to:

  • Allow an object to be placed on a marker
  • Make an object immediately snap to a marker

Requirements

  • Unity for Windows
  • Meta SDK Installer
  • One pair of Meta glasses
  • Printer

This tutorial also assumes that you have already completed the following:

  • Making a MetaBody

Steps

Setting up

  1. Plug in and set up your Meta glasses
  2. Run Unity
  3. Open the 'Meta' project
  4. Open your 'Hello MetaWorld' scene

Printing a marker

  1. Open the AprilTagMarkers0-11.pdf file inside your Meta project
    1. Meta\MetaDocs\AprilTagMarkers0-11.pdf
  2. Print out the marker PDF file
  3. Place the markers on a surface or object that you want to be tracked

Making a GameObject 'Placeable' on a marker

  1. Select the 'Cube' from your 'Hello MetaWorld App' GameObject
    1. Expand the 'Hello MetaWorld App' GameObject in the Hierarchy window by clicking the triangle next to the name
    2. Select the 'Cube'
  2. Check the 'Marker Target' and 'Placeable' checkbox and set 'Marker ID' to -1 in the MetaBody script in the Inspector window
    Checking the Marker Target box in MetaBody
  3. Put on your Meta glasses
  4. Save the scene
  5. Turn your head and look at the marker that you printed
  6. Run the scene by clicking the Play button
    The cube and marker in the Game View
  7. Grab the 'Cube' and move it towards the marker
  8. Release the 'Cube' and watch it snap to the position of the marker
  9. Stop the scene by clicking the Play button again

Making a GameObject immediately snap to a marker

  1. Select the 'Cube' from your 'Hello MetaWorld App' GameObject
  2. Set 'Marker ID' to the ID of your printed marker (eg. 0) in the MetaBody script in the Inspector window
  3. Put on your Meta glasses
  4. Save the scene
  5. Turn your head and look at the marker that you printed
  6. Run the scene by clicking the Play button
    1. The cube should immediately snap to the position of the marker if it is in view
  7. Stop the scene by clicking the Play button

Making a GameObject remember its last marker position

  1. Select the 'Cube' from your 'Hello MetaWorld App' GameObject
  2. Check the 'Persistent' checkbox in the MetaBody script in the Inspector window
  3. Set 'Marker ID' to -1 and click the 'Save' button next to the 'Persistent' checkbox to reset the last remembered marker
  4. Put on your Meta glasses
  5. Save the scene
  6. Turn your head and look at the marker that you printed
  7. Run the scene by clicking the Play button
  8. Grab the 'Cube' and move it towards the marker
  9. Release the 'Cube' and watch it snap to the position of the marker
  10. Stop the scene by clicking the Play button
  11. Run the scene by clicking the Play button
    1. The cube should remember the marker that it was on (the "Saved ID") and immediately snap to its position

Removing Marker Target tracking from a GameObject with MetaBody

  1. Select the 'Cube' from your 'Hello MetaWorld App' GameObject
  2. Uncheck the 'Marker Target' checkbox in the MetaBody script in the Inspector window
  3. Save the scene

Conclusion

By checking the 'Marker Target' box, the GameObject will stay attached to the marker with the ID specified in the 'Marker ID' box in the MetaBody script. (-1 is the default value, and this will cause it to remain unsnapped until you drag the object near a marker)

Checking 'Placeable' will allow you to switch the marker an object is attached to when it is grabbed/pinched and moved. Checking 'Persistent' will allow you to make a GameObject remember which marker it was last attached to when running the scene.

For further reference on how to use Marker Targets, refer to the MetaBody Component Guide.
