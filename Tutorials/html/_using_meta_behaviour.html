Writing a MetaBehaviour

To handle events and specifications of gestures

Overview

This guide will show you how to use the MetaBehaviour class (an extension of MonoBehaviour) to handle and react to gesture-based events (eg. grabbing an object).

By the end of this guide, you will be able to:

  • Create a new script that inherits from the MetaBehaviour class
  • Handle a grab event to make a cube change colour

Requirements

  • Unity for Windows
  • Meta SDK Installer
  • One pair of Meta glasses

This tutorial also assumes that you have already completed the following:

  • Making a MetaBody

Steps

Setting up

  1. Plug in and set up your Meta glasses
  2. Run Unity
  3. Open the 'Meta' project
  4. Open your 'Hello MetaWorld' scene

Adding and editing a script on a GameObject to use MetaBehaviour

  1. Select the 'Cube' from your 'Hello MetaWorld App' GameObject
    1. Expand the 'Hello MetaWorld App' GameObject in the Hierarchy window by clicking the triangle next to the name
    2. Select the 'Cube'
  2. Uncheck the 'Jaysify' checkbox in the MetaBody script in the Inspector window
  3. Add a new C# script called 'MyCubeScript' to the 'Cube'
    1. Click Add Component at the bottom of the Inspector window
    2. Select New Script >
    3. Type in 'MyCubeScript' with language 'CSharp' and click 'Create and Add'
      Adding MyCubeScript
  4. Open MyCubeScript for editing by double-clicking the 'MyCubeScript' script property in the Inspector window
  5. Add Using Meta; below the Unity namespace libraries
    1. Add the following lines:
      using UnityEngine;
      using Systems.Collections;
      using Meta;
  1. Edit the script so that the MyCubeScript class inherits from MetaBehaviour instead of MonoBehaviour
    1. Find the following line:
      public class MyCubeScript : MonoBehaviour {
    2. Change it to:
      public class MyCubeScript : MetaBehaviour {
  2. Add a OnGrab() and OnGrabRelease() method to handle a grab event and make the cube turn red when grabbed and white when released
    1. Add the following lines inside the MyCubeScript class below the Update() function:
      public void OnGrab() {
          renderer.material.color = Color.red;
      }

      public void OnGrabRelease() {
          renderer.material.color = Color.white;
      }
  3. Save your changes to the script
  4. Put on your Meta glasses
  5. Save the scene and click the Play button to run the scene
  6. Turn your head to look for the 'Cube' (or push F4 to recentre the Game View)
  7. Grab the 'Cube' and it should change colour from white to red
Cube is white when ungrabbed
Cube is red when grabbed

Conclusion

You have now learned how to use the MetaBehaviour class. There are many other handlers available, including OnTouch(), that will allow you to easily deal with any gestural interaction.
