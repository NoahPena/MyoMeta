Meta.Gaze Class Reference

Description

Gaze implements gaze-based input by raycasting objects in front of the center of the screen.

To make something respond to gaze, make it MetaBody.Gazeable. To make something respond to gaze dwell, make it MetaBody.gazeableDwellable.
