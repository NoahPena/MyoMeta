Meta.MetaBody Class Reference

Description

Adds dynamic transforming options to make MetaObjects that can orbit and be interacted with to move them between various states via gestures.

MetaBody component makes a game object Meta-aware, with gestural, positional and visual feedback integration with hand input. Gestures lets you turn on Grab and Pinch and set which interactions they trigger.

Guide

The MetaBody Component Guide section of the Meta App Programming Guide explains in-detail how to use MetaBody.

Tutorial

To learn how to use MetaBody, follow the Making a MetaBody tutorial.
