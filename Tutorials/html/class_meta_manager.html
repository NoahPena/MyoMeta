MetaManager Class Reference

Description

Class for handling most Meta behaviours of MetaObjects. Scripts inherited from MetaBehaviour add themselves to MetaManager's registry to receive events. MetaManager can inform other systems about what events the currently existant MetaObjects can handle so unnecessary event checking can be skipped.
