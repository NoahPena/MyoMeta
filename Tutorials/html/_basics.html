Meta.exe

The Meta.exe launches the Meta operating system. The application launches when you install the SDK and you can return to it by launching it from the Desktop App folder or Start menu You can also access it through the installed path, ie. Default path C:\Program Files (x86)\Meta\Meta SDK\Meta.exe.

  • Meta.exe starts in the Meta Welcome App. It introduces users to Meta and checks the connection status of the Meta glasses hardware.
  • Meta.exe includes the Meta SDK Guide App which introduces each capability of the API in an interactive form you can try with links to example code and documentation.

Meta Guide

The Meta Guide folder contains the HTML help guide (you are reading it now). You can access Meta Guide from within the Unity editor by clicking Unity Menu > Help > Meta Documentation.

Meta.unitypackage

To create new Meta projects:

  1. Select File > New Project... in the Unity editor.
  2. Under the Import the following packages: section, check the Meta.unitypackage box. Leave the other boxes unchecked unless you require that functionality.
  3. Choose your project location, then click Create.

To import the Meta package into an existing Unity project:

  1. Make a backup of the existing project by copying the project folder to another location.
  2. Select Assets > Import Package > Meta in the Unity editor.
    • If Meta does not appear in the Import Page list, select Custom Package... and find C:\Program Files (x86)\Meta\Meta SDK\Meta.unitypackage.
  3. Click Import. This will add all the necessary files to begin developing a Meta App.

When upgrading:

  1. Install the latest MetaInstallerDotExe.
  2. Make a backup of the existing project by copying the project folder to another location.
  3. Check the Release Notes for upgrade guidance.
  4. Import the latest Meta.unitypackage by following the steps in the section above.
  5. Confirm that everything still works.

Quick Start Instructions

You can get started by following Meta Tutorials. Here are some quick start instructions on how to do common tasks:

How do I create a Meta project in Unity?

  1. Select File > New Project... in the Unity editor.
  2. Under the Import the following packages: section, check the Meta.unitypackage box. Leave the other boxes unchecked unless you require that functionality.
  3. Choose your project location, then click Create.

Refer to the MetaDotunitypackage section of this page for importing into existing projects and upgrading.

How do I make my first working Meta scene?

  1. Select File > New Scene in the Unity editor.
  2. In Hierarchy, delete the Main Camera.
  3. In Project, drag the Metaworld prefab into the Hierarchy.
  4. Click the Play button.

See the Hello MetaWorld tutorial for a more detailed walkthrough.

How do you make objects respond to grab gestures?

  1. Drag a MetaCube primitive into the scene or add the MetaObject component onto an object.
  2. Hit play. (MetaBody.gestureable, MetaBody.grabbable and MetaBody.moveable are on by default).

See the Making a MetaBody tutorial for a more detailed walkthrough.

How do I track objects to surfaces?

  1. Drag the MetaBody component onto an object.
  2. Check MetaBody.canvasTarget.
  3. Click the Play button.

See the Tracking a CanvasTarget tutorial for a more detailed walkthrough.

How do I track objects to float around me?

  1. Drag MetaBody component onto an object.
  2. Check Locking and check Orbital.
  3. Click the Play button.

User Settings

There are a number of settings that you are able to customize in order to achieve an optimal viewing and interactive experience. These changes can be saved to a user profile using the Meta User Settings Window which can be found in the Meta menu.

User IDs

There are two ways to specify a user ID to create or load a user profile:

  • Logging in using MetaLogin - You can log in to your Meta account and save your user ID through the MetaLogin panel in MetaWorld.
  • Use a Local ID - In the Meta User Settings Window, there is an option to save your own Local ID. By default, your user ID is MetaUser.

Once you have a User ID or Local ID, you will have a User Settings file that is associated with your ID.

IMU Model

The IMU in the Meta 1 is used for rotational head tracking. Prototype glasses shipped before July 11 2014 contain the HID IMU (MPU9150HID), whereas glasses shipped after this date contain the Serial IMU (MPU9150Serial). If no IMU is selected, the Meta applications will default to the Serial IMU.

If your IMU has been correctly selected and saved in your User Settings but your head tracking is still not working, refer to the Troubleshooting Guide.

Rendering Profile

The rendering profile contains the Meta 1 cailbration parameters that match your vision. To run the calibration process, open Calibration.exe or the MetaWelcomeScene.unity scene in your project and click the Calibration button to begin the calibration process.
