Hardware Setup

Before you can use the Meta SDK, you must ensure your Meta hardware is properly connected.

  1. Unzip the Meta 1 Development kit case.
  2. Connect the Meta 1 Glasses to the Control Box via the two USB ports and Display Cable port.
    • These USBâ€™s connect the IMU and Camera to the control box.
  3. Once the Meta 1 glasses are connected to the Control Box, connect the Control Box to power and your Computer:
    • Plug the Mini-USB cable into the Control Box and connect the other end to your computer.
    • Plug in the included Power Adapter into the Meta Control Box and a power outlet.
    • Plug the HDMI cable into the Control Box and your Computer
    • There should also be a green light on the front of the Control Box indicating it is On.
  4. The Meta 1 Development Kit is automatically On when powered. The Power button when pressed will alternate the Meta 1 between On and Sleep mode.
  5. Adjust the Meta 1 Glasses head straps to fit comfortably on your head. There is an extra support strap which goes over your head to offer additional support.

Extended Information

  1. Meta 1 Development Kit comes with two lens covers; the FOV* Expander Lens and the Shade Lens. Choose the lens cover you will be working with before putting the Meta 1 Glasses on. Turn the Meta 1 Glasses upside-down, slide open the small tabs to release the lens cover, switch out the new cover, slide the tabs closed to secure the lens cover in place.
  2. Configure your computer to the appropriate settings for using the Meta 1 Development Kit.
    • The Meta 1 glasses act as an external Monitor, therefore use mirrored mode or extended monitor.
    • The HDMI cable connects the audio to your Meta 1 Glasses so configure your computer accordingly.
  3. Use the 2D/3D button to toggle between 2D and 3D modes.

Software Installation

Before installing the Meta SDK software, make sure your computer meets the Minimum Requirements. Only Windows OS is supported - we do not support running the Meta SDK in a virtual machine (eg. VirtualBox or Parallels).

NOTE: If you have a Mac computer, you must first use Apple's BootCamp utility to partition it to dual boot a Windows 8 operating system.

To install the SDK on Windows:

  1. Download and install the latest version of Unity 4.6 or newer.
  2. Download the latest version of the Meta SDK (MetaInstaller.exe) from our Developer Portal.
  3. Run the MetaInstaller.exe setup file.
  4. Make sure your glasses are turned on and set up correctly.
  5. Go to your Apps folder shortcut on the desktop or search for it in the location where you installed the Meta.
  1. Explore the Apps!
    • Get to know Meta features by running through the Meta.exe
    • Calibrate for yourself with Calibrate.exe
    • Run through all features available in the SDK with the Meta SDK App.exe
  2. Use your fingers to touch the buttons on the screen.

Control Box Features

Control Box Features

The Meta 1's Control Box has a few buttons you will need to understand and use:

Meta 1 Control Box Button Function
2D/3D Toggle Switch between 2D desktop mode and stereoscopic split-screen
Brightness Adjust the brightness of the display
Power Hold to turn on

LED light colors

The Meta 1 Control Box has an LED on the front which indicates the current state that the box is in:

Light Color Meaning
Red AC adapter connected, control box switched off
Green Control box switched on, USB ports active
Orange Control box switched off, USB ports active
Blue Control box switched on, USB ports inactive in sleep mode

Integrated USB hub

The Meta 1 Control Box includes an integrated USB hub. To use the hub, connect the Mini USB cable from the control box to your computer, and the two USB cables from your glasses into the control box.

NOTE: DO NOT plug in both ends of the Mini USB to USB cable into the control box. This may cause a short circuit and is NOT covered by warranty.

HDMI input

There is a HDMI port on the control box that acts as the display source for the Meta 1 glasses. This should be connected to your PC with the included HDMI cable.

Early Dev Kit (pre-July) Control Boxes

On early Meta 1 Developer Kit Control Boxes, there is an internal battery and an LED light indicating its status. There is also a sleep toggle button on the glasses cable that can turn the display on and off instantly.

LED light colors

The early dev kit control boxes also have an LED indicator that conveys the battery status of the box:

Light Color Meaning
Red Out of battery, needs to be charged up before use
Green Charged, ready to use
Orange Turning on, in use
Blue In stand-by mode

Other Notes

Configuring the Meta 1 Extended Monitor Feature

It is easiest to start by using the "Duplicate your display" option so that what you see on your desktop and your glasses is the same. When you hit Play in Unity, you can select the Maximize On Play option to make the Game View take up most of the window.

However, does not fill the entire screen and, unless you set the Windows taskbar to auto-hide, you may find your view is cluttered by taskbar icons on the bottom of the screen and Unity editor buttons on the top.

Computer Monitor

  1. In Windows, right-click the desktop and click "Screen Resolution".
  2. Set the "Multiple displays" option so that the glasses are extending the desktop instead of duplicating it.
    • Resolutions on the glasses work best at 16:9 Ratio

Unity

  1. In Unity, the Panel menu is at the right side of each panel. It is an icon of a triangle and three lines:
    The Unity panel menu
  2. Click the Panel menu and choose Add Tab > Game.
  3. Drag the extra Game View tab out of its parent window and resize it to be a small rectangle.
  4. Drag the floating Game View onto the desktop of the Meta glasses.
  5. Click on the Maximize button on the top right.
    1. You will need to look through the glasses to do this. To make it easier, set the glasses' physical 2D/3D toggle switch to 2D.
  6. Click on Play in Unity and check if the Game View works.
    1. Keyboard input may not work until you click on the Game View. You may need to move your mouse cursor off of your main screen and onto the glasses display to click on it.
