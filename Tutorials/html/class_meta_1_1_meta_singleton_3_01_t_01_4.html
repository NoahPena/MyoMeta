Meta.MetaSingleton< T > Class Template Reference

Description

The classes that inherit this object will have a GameObject with the script on it made automatically.

A MetaSingleton creates a static Instance variable that refers to the single instance of the class at Unity Awake(). You can implement a static pattern without getting references to the singleton by calling ClassName.Instance.member. Note however that unlike a true static class, the instance doesn't exist till Unity plays so it can't be directly accessed by things like Editor inspectors.

This class replaces the old MonoSingleton.
