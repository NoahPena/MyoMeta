Meta.Localizer Class Reference

Description

The localizer is a base abstract class for all the localization Objects.

This class is a parent to all localizer. and enables Meta Localization to switch between different localization module. including IMU, mouse, Slam etc. Also the target Game Object is the Game object that is affected by this Localizer module. This target Game object is the game object that will be moved by the localizer. Also this game object will be transfered between the localizers when they are switched in the MetaLocalization Module. If No default target game Object is set, the MetaFrame Game object is used.
