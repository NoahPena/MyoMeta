Making a MetaBody

Making a MetaBody

Overview

This guide will show you how to use the MetaBody script to allow the objects in your scene to be manipulated by gestures (eg. grab, pinch, poke).

By the end of this guide, you will be able to:

  • Add the MetaBody scripts to objects
  • Enable gesture interactions on the MetaBody script
  • Make objects fade in with a Jaysify effect

Requirements

  • Unity for Windows
  • Meta SDK Installer
  • One pair of Meta glasses

This tutorial also assumes that you have already completed the following:

  • Hello MetaWorld

Steps

Setting up

  1. Plug in and set up your Meta glasses
  2. Run Unity
  3. Open the 'Meta' project
  4. Open your 'Hello MetaWorld' scene
  5. Delete the 'MetaCube' from your 'Hello MetaWorld App' GameObject
    1. Expand the 'Hello MetaWorld App' GameObject in the Hierarchy window by clicking the triangle next to the name
    2. Right-click and delete the 'MetaCube' GameObject
Deleting MetaCube

Adding the MetaBody script to a GameObject

  1. Create a new cube as a child of 'Hello MetaWorld App' at position (0, 0, 0.4) with scale (0.1, 0.1, 0.1)
    1. GameObject > 3D Object > Cube
    2. Drag the 'Cube' that appears in the Hierarchy window onto 'Hello MetaWorld App'
    3. Select the 'Cube' in the Hierarchy window
    4. Modify the values in the Inspector window so that the position is (0, 0, 0.4) and the scale is (0.1, 0.1, 0.1)
  2. Add the MetaBody script to the 'Cube'
    1. Select the 'Cube' in the Hierarchy window
    2. Click Add Component at the bottom of the Inspector window
    3. Type in MetaBody and select it from the search results
  3. Check the 'Gestures', 'Grabbable' and 'Move' checkboxes in the Inspector window
    MetaBody script on Cube
  4. Save the scene
  5. Put on your Meta glasses
  6. Click the Play button to run the scene
  7. Turn your head to look for the 'Cube' (or push F4 to recentre the Game View)
  8. Grab the 'Cube' and move it around with your hand
  9. Stop the scene by clicking the Play button again

Adding the Jaysify effect

  1. Select the 'Cube' in the Hierarchy window
  2. Check the 'Jaysify' checkbox in the MetaBody script in the Inspector window
    Jaysify checkbox
  3. Save the scene and click the Play button to run the scene
  4. Watch as the cube fades in with the Jaysify effect
  5. Grab the 'Cube' again and move it around with your hand
  6. Stop the scene by clicking the Play button again

Removing the Jaysify effect

  1. Uncheck the 'Jaysify' checkbox in the MetaBody script in the Inspector window
  2. Save the scene

Conclusion

Now you know how to make any object support gesture interactions! Adding the MetaBody script to a game object makes it Meta-aware, with gestural, positional and visual feedback integration with hand input. The 'Jaysify' option will make your object fade in with a sci-fi line build-in effect.

For further reference on the components of MetaBody, refer to the MetaBody Component Guide.
