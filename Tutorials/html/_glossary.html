Glossary

Glossary of Terms

Term Definition
AUGMENTED REALITY
augmented reality (AR) Mediating what we see by placing graphics over it that enhance, add or diminish information.
rectification When what you see through the glasses matches what you see in the real world and what the depth sensors can track about the physical world.
wearable computer A computer you wear so that you don't have to sit down to operate it.
additive A way of adding graphics to reality by brightening an area with projected light. (rather than darkening it)
depth sensor A camera that perceives the surface of objects by bouncing infrared light off of them.
time-of-flight A type of depth sensor that measures the round-trip time it takes infrared light to bounce off a surface and return to the sensor.
RGB feed Red Green Blue camera feed, used to refer to the visible light camera feed that can be enabled in the background.
head mounted display (HMD) A display you wear on your face, used for VR and AR purposes.
Field of View (FOV) Area of view seen through the Meta glasses. For more info, refer to Field of View.
 
UNITY
Unity A 3D game engine used to write Meta Apps.
C# A programming language used with Unity to write Meta Apps.
.unitypackage A set of files that can be imported into a Unity project.
 
META
Finger Indicator The circles that follow the fingertips on each of your hands. (F7 to toggle - refer to Keyboard Shortcuts)
GestureManager (Meta GestureManager) Core component that communicates with MetaBody to keep track of MarkerTargets, CanvasTargets and gesture interactions.
Hands Core gesture controller that communicates with the C++ gesture libraries to get the position and state of the hands.
Hand Point Cloud The augmented representation of your hand within a Meta App. (F6 to toggle - refer to Keyboard Shortcuts)
Jaysify/ Jaysification Unity effect of graphic rendering made by Jayse Hansen our lead UI designer and creator of Iron Man graphics
MetaBasic Combination of essential concepts you will need to know to use Meta. For more info, refer to Meta Basics.
MetaBehaviour New scripts you write that allow you to handle Unity and Meta gesture-based events.
MetaBody Script component that you can add to any object to allow it to have gesture interaction support, marker/canvas tracking ability and other Meta method functionality. For more info, refer to MetaBody Component Guide.
    MarkerTarget The component that allows you to track an object to a marker.
    CanvasTarget The compenent that allows you to track an object to a markerless surface.
MetaCamera Special camera set up for stereoscopic rendering and head tracking motion. Replaces the Main Camera in Unity.
    Rectified Stereo mode Stereo, 1:1 true-scale view for creating Meta 1 UIs, and zoomed use of future Meta UIs.
    Wide-angle Stereo mode Stereo, wide view for multi-tasking and future Meta UI prototyping.
    Wide-angle Mono mode Single, wide view for editing Meta UIs without the glasses connected and demoing functionality to observers.
MetaGraphicalUserInterface: MGUI Graphical User Interface framework based on Unity UI. Provides GUI creation tools, including buttons and panels.
MetaPrimitives Example GameObjects that demonstrate the basic functionality achievable with the Meta SDK.
MetaWorld Core prefab that you place in a Unity scene to replace Main Camera with stereoscopic MetaCamera and Hands system. Makes the scene a Meta application scene.
