Meta.MetaStore Class Reference

Description

MetaStore connects to the MetaStore server and handles user and developer data.

  • UserLogin() provides authentication services to the MetaLogin app. MetaSpace calls on MetaStore to handle cloud data services. MetaSpace can be used offline without logging in for rapid development.
