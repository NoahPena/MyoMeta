bool MetaSmoothing.AdaptiveKalmanEnabled = false

Enables motion-dependent Kalman filtering, so that filtering is intense when there is no motion and high-pass when the target is moving. This modifies the velocity of Kalman filtering. It is a hack until a more complex method is employed.

float MetaSmoothing.SpeedToKalmanVelocityMultiplier = 100.0f

This multiplier converts the object's speed to Kalman velocity. Kalman velocity is merely a smoothing parameter. It is not necessarily a meaningful parameter. The largest it is, the less smoothing happens. TODO: parametrize using meaningful parameters.
