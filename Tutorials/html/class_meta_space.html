MetaSpace Class Reference

Description

MetaSpace provides an operating system environment framework for Meta applications.

MetaSpace calls on MetaStore for cloud data, user login and app management. MetaSpace stores defafult MetaVectorStyle settings for VectorMeshObjects attached using metaBody.jaysify.

enum MetaSpace.States

Enumerator
Menu 

Navigating a menu overlay freezes input to all other parts of apps.

App 

Focusing on a particular app allows input to that app.

Store 

Accessing the store overlay freezes input to all other apps.

Settings 

Accessing settings

Sleep 

Low-power mode suspends background processes and wakes them back up after sleep

Shutdown 

Shutdown closes all processes

Startup 

Startup mode ends when all system apps have started.
