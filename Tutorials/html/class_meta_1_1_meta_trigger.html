Meta.MetaTrigger Class Reference

Description

MetaTrigger is a general-purpose trigger script to make an object trigger events based on finger, reticle, mouse or SendMessage input.

The action selector determines what the MetaTrigger will do on receiving a DoTrigger event or having its finger, reticle or mouse handling fire.

trigger a MetaTrigger by SendMessage, send it a "DoTrigger" message. To make MetaTrigger activate or deactivate an object, To make MetaTrigger send a message to another object which could have other MetaTriggers, choose Message type and pick a messageName and optionally a messageValue to pass to the handler function with the same name in the target object's scripts.

Here's an example of sending a DoTrigger message to a MetaTrigger manually from code:

public void OnFingerEnter()
{
GameObject.Find("TheMetaTriggerObject").SendMessage("DoTrigger");
}
