Learning Unity Guide

Learning Unity

If you are new to Unity, here are some recommended starting points:

  1. Unity Editor Interface video tutorial: Get comfortable with the camera controls and editor tools in Unity by watching and review any concepts you want to clarify in the Unity Editor Interface Docs.
  2. Unity Scripting Tutorial Videos: Dive into learning C# scripting. Great even if you are new to programming as they start from scratch and for experienced programmers the videos explain how familiar programming concepts apply within Unity.
  3. Unity Interactive Tutorials Series: Learn how to create Unity scenes from within Unity with this guided walkthrough.

Unity Update Sequence

Unity's event sequence is an important thing to understand. Meta's events take place in combination with these and your logic may fall somewhere in between these events in a way that influences how things work.

Building standalone executables

A Meta App can be built as a standalone executable application. To do this:

  1. In your Unity scene, select File>Build Settings
  2. In the Build Settings window, select PC, Mac and Linux Standalone
  3. Set the target platform as Windows and architecture as x86
  4. Click Player Settings... in the bottom left
  5. In the PlayerSettings Inspector window, find the Other Settings section and make sure the Api Compatibility Level is set to .NET 2.0
  6. Return to the Build Settings window, and click Build
  7. Select a location and name for your build executable and data folder, and then click Save
